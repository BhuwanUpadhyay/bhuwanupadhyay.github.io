Expose spring boot microservice with ingress using helm

In k8s deployment, ingress exposes HTTP and HTTPS routes from outside the cluster to services within the cluster. In this article, I will take you through how to expose spring boot microservice for the outside world in k8s deployment using ingress.

This example needs kubectl minikube helm command-line tools in your machine.
